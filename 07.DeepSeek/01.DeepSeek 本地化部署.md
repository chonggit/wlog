# DeepSeek 本地化部署

## 部署方式

1. **直接使用 Ollama**：这是最快捷的本地部署方式，适合开发者快速测试模型

2. **源代码部署**：若需定制化环境，可参考 GitHub 仓库的安装指南构建镜像


### 直接使用 Ollama

部署环境：Ubuntu 18.04.5

安装 Ollama

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

启动 deepseek-coder-v2:16b 模型服务

```bash
ollama run deepseek-coder-v2:16b
```
