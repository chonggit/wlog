# 软件项目实施（开发）转运维交付标准

# 一、代码与制品交付（核心）
1. **软件产品/构建产物**
   - Jar/War/Bundle/dll/镜像包/软件安装包
2. **源代码（如有）**
   - 完整可编译源码（Git 仓库、分支、Tag）
   - 版本说明、Commit 日志、ChangeLog
3. **容器/云原生制品（如有）**
   - Dockerfile、docker-compose.yaml
   - Helm Chart、K8s YAML（Deployment/Service/Ingress/ConfigMap等）

# 二、环境与配置交付
1. **环境清单**
   - 开发/测试/生产系统配置要求（CPU、内存、硬盘）
   - 开发/测试/生产环境信息（IP、域名、端口、路径）
2. **配置文件**
   - application.yml、bootstrap.yml、.env、properties、appsettings.json、web.config 等
3. **环境依赖版本**
   - JDK/Node/Go/.NET、Nginx、Redis、MySQL、MQ、ES、MSSQL、Linux、Windows等版本要求

# 三、数据库交付（如有）
1. **数据库脚本**
   - DDL：建表、索引、约束、初始化语句
   - DML：基础数据字典、枚举数据
   - 增量更新 SQL、回滚 SQL
2. **数据库文档**
   - ER 图、数据字典
3. **迁移/初始化方案**
   - 数据初始化顺序、历史数据迁移脚本

# 四、部署与运维手册
1. **部署文档**
   - 部署步骤、启动/停止/重启命令（或操作说明）
   - 端口占用、路径规划、日志路径
2. **配置说明**
   - 核心配置项含义
3. **监控指标**
   - CPU、内存、硬盘

# 五、运行维护类交付物
1. **日志规范**
   - 日志路径、格式、级别、切割策略
2. **备份策略**
   - 数据库备份、文件备份、备份周期

# 六、账号与权限交付
1. **应用账号清单**
   - 管理员账号、业务账号、第三方系统账号
2. **权限矩阵**
   - 数据库账号、中间件账号、服务器账号、云资源权限
